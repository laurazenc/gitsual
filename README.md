## Usage
This output files would be generated in the `dist` folder
```bash
yarn dev
```


## Packaging
The installer would be in the `releases` folder.

```bash
yarn dist
```


This would generate an installer for `m`ac, `w`indows and `l`inux

```bash
yarn dist mwl
```
